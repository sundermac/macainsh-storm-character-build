# Ponyo Character Build - HW #2

The MMED-1054 HW#2 is a one-page character build inspired by the Studio Ghibli anime movie, *Ponyo*. The page is styled to match the anime's theme and has character profiles for Ponyo, Sosuke, Lisa, and Fujimoto. Clicking on a character opens a lightbox containing a detailed description of them.
